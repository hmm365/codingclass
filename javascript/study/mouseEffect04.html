마우스 이펙트4

이번 마우스이펙트는 마우스가 움직일때 이미지가 움직입니다.
centerPageX = window.innerWidth / 2 - mousePageX; 을 이용하여 가운데 사진의 좌표값을 지정합니다. 그후 GSAP를 이용하여 효과를 씁니다.

JS 설명

gsap를 써서 마우스커서가 움직이게 작동했습니다.

코드 보기 
gsap.to(imgMove, {
    duration: 0.5,
    left: centerPageX / 20,
    top: centerPageY / 20,
});
