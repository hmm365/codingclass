1부터 100까지 출력하기

1.1 while문을 이용해서 1부터 10까지 출력하시오!

let result = document.querySelector(".sample01_result");
let i=1; 
while(i<=10){
    result.textContent += i + " ";
    i++;
}

1.2 for문을 이용해서 1부터 10까지 출력하시오!

let result = document.querySelector(".sample02_result");
for(let i=1; i<=10; i++){
    result.innerHTML += i + " ";
}

1.4 for문을 이용해서 1부터 100까지 출력하시오!(1~100까지 더한수)

let result = document.querySelector(".sample04_result");
let sum = 0;
for(let i=1; i<=100; i++){
    sum += i
}
