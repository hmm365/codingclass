HTML 태그 검색하기

전체 속성 갯수 : 0
  • a : a 태그는 다른 페이지 이동을 설정합니다.
  • abbr : abbr 태그는 줄임말이나 머리글자를 표현하는 사용합니다.
  • acronym : acronym 태그는 준말 또는 머리글자를 나타냅니다.
  • address : address 태그는 가까운 HTML 요소의 사람, 단체, 조직 등에 대한 연락처 정보를 나타냅니다.
  • applet : applet 태그는 자바 애플릿을 보이게 하는 곳에 쓰입니다.
  • area : area 태그는 이미지의 핫스팟 영역을 정의하고 하이퍼링크 (en-US)를 추가할 수 있습니다. map 요소 안에서만 사용할 수 있습니다
  • article : article 태그는 문서, 페이지, 애플리케이션, 또는 사이트 안에서 독립적으로 구분해 배포하거나 재사용할 수 있는 구획을 나타냅니다.
  • aside : aside 태그는 문서의 주요 내용과 간접적으로만 연관된 부분을 나타냅니다. 주로 사이드바 혹은 콜아웃 박스로 표현합니다
  • audio : audio 태그는 문서에 소리 콘텐츠를 포함할 때 사용합니다. src 특성 또는 (en-US) 요소를 사용해 한 개 이상의 오디오 소스를 지정할 수 있으며, 다수를 지정한 경우 가장 적절한 소스를 브라우저가 고릅니다.
  • b : b 태그는 자의 주의를 요소의 콘텐츠로 끌기 위한 용도로 사용합니다. 그 외의 다른 특별한 중요도는 주어지지 않습니다.
  • base : base 태그는 문서 안의 모든 상대 URL이 사용할 기준 URL을 지정합니다. 문서에는 하나의 요소만 존재할 수 있습니다.
  • bdi : bdi 태그는 BDI는 Bi-Directional Isolation을 의미하며, 주위의 텍스트들과는 다른 방향으로 써지는 텍스트 영역을 정의할 때 사용합니다.
  • bdo : bdo 태그의 BDO는 Bi-Directional Override를 의미하며, 자식 요소의 텍스트 방향성(text directionality)을 정의할 때 사용합니다.
  • bgsound : bgsound
  • big : big
  • blink : blink
  • blockquote : blockquote
  • body : body 태그는 해당 HTML 문서의 텍스트, 하이퍼링크, 이미지, 리스트 등과 같은 모든 콘텐츠를 포함하는 영역을 정의할 때 사용합니다.
  • br : br 태그는 텍스트 내의 줄바꿈(line-break)을 정의할 때 사용합니다. _
  • button : button 태그는 클릭할 수 있는 버튼을 정의할 때 사용합니다.
  • accent-color : accent-color 는 특정 요소에 색상을 지정합니다.
  • align-content : align-content 는 콘텐츠 아이템의 상하관계 정렬 상태를 설정합니다.
  • align-items : align-items 는 콘텐츠 아이템의 내부 상하관계 정렬 상태를 설정합니다.
  • align-self : align-self 는 아이템의 교차축 정렬 정하는 속성 입니다.
  • all : all 는 요소의 속성을 초기화 또는 상속을 설정합니다.
  • animation : animation 는 HTML 요소에 적용되는 CSS 스타일을 다른 CSS 스타일로 부드럽게 변화시킵니다.
  • animation-delay : animation-delay 는 적용 대기 시간을 지정합니다.
  • animation-direction : animation-direction 는 애니메이션의 진행 방향을 설정합니다.
  • animation-duration : animation-duration 는 실행된 애니메이션 진행 시간을 지정하는 속성입니다.
  • animation-fill-mode : animation-fill-mode 는 애니메이션 실행 이전이나 이후에 애니메이션 효과를 표시할지의 여부를 지정하는 속성입니다.
  • indexOf() : indexOf() 메서드는 문자열에서 특정 문자의 위치를 찾고 숫자를 반환합니다.
  • slice() : slice() 메서드는 문자열에서 원하는 값을 추출하여 문자열을 반환하는 메서드 입니다.
  • substring() : substring() 메서드는 문자열에서 원하는 값을 추출하여 문자열을 반환하는 메서드 입니다.
  • substr() : substr() 메서드는 문자열에서 원하는 값을 추출하여 문자열을 반환하는 메서드 입니다.
  • split() : split() 메서드는 문자열에서 원하는 값을 추출 배열로 변환합니다.
  • trim() : trim() 메서드는 문자열의 앞뒤 공백을 제거합니다.

//선택자
const searchBox = document.querySelector(".search__box input"); //검색 박스
const searchList = document.querySelectorAll(".search__list ul li"); //리스트 목록
const searchInfo = document.querySelector(".num")
searchInfo.textContent = searchList.length;
searchBox.addEventListener("keyup", () => {
const searchWord = searchBox.value; //사용자가 입력한 키워드

searchList.forEach((el) => {
const cssName = el.dataset.name;

if (cssName.indexOf(searchWord)) {
el.classList.add("hide");
} else {
el.classList.remove("hide"); //indexOf() 값이 0이되는 값만 hide를 지워줍니다.
}

});
})
