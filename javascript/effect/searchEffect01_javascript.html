indexOf()를 이용하여 JAVASCRIPT 메서드 검색히가

JAVASCRIPT 메서드 검색히가

  • indexOf() : indexOf() 메서드는 문자열에서 특정 문자의 위치를 찾고 숫자를 반환합니다.
  • slice() : slice() 메서드는 문자열에서 원하는 값을 추출하여 문자열을 반환하는 메서드 입니다.
  • substring() : substring() 메서드는 문자열에서 원하는 값을 추출하여 문자열을 반환하는 메서드 입니다.
  • substr() : substr() 메서드는 문자열에서 원하는 값을 추출하여 문자열을 반환하는 메서드 입니다.
  • split() : split() 메서드는 문자열에서 원하는 값을 추출 배열로 변환합니다.
  • trim() : trim() 메서드는 문자열의 앞뒤 공백을 제거합니다.
