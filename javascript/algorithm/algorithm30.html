문자열 다루기 기본

문자열 s의 길이가 4 혹은 6이고, 숫자로만 구성돼있는지 확인해주는 함수, solution을 완성하세요.
예를 들어 s가 "a234"이면 False를 리턴하고 "1234"라면 True를 리턴하면 됩니다.

s는 길이 1 이상, 길이 8 이하인 문자열입니다.
s는 영문 알파벳 대소문자 또는 0부터 9까지 숫자로 이루어져 있습니다.

입출력 예

s return
"a234" false
"1234" true 
function solution(s) {
    // return s.includes(s.match(/[a-zA-Z]/)) == true ? false : true; 문자열을 찾아버렸는데 오류가뜸
    return s.includes(s.match(/^\d{6}$|^\d{4}$/));
}

총평 : 정규식으로 문자열을 찾으면 오류가뜨는건 아마 숫자도 문자열로 취급되어있어서 그런것 같습니다.
