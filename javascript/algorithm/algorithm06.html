없는 숫자 더하기

0부터 9까지의 숫자 중 일부가 들어있는 정수 배열 numbers가 매개변수로 주어집니다. numbers에서 찾을 수 없는 0부터 9까지의 숫자를 모두 찾아 더한 수를 return 하도록 solution 함수를 완성해주세요.

1 ≤ numbers의 길이 ≤ 9
0 ≤ numbers의 모든 원소 ≤ 9
numbers의 모든 원소는 서로 다릅니다.

입출력 예

numbers result
[1,2,3,4,6,7,8,0] 14
[5,8,4,0,6,7,9] 6 
입출력 예 #1

5, 9가 numbers에 없으므로, 5 + 9 = 14를 return 해야 합니다.

입출력 예 #2
    
    1, 2, 3이 numbers에 없으므로, 1 + 2 + 3 = 6을 return 해야 합니다.

없는 숫자 더하기

function solution(numbers) {
    var answer = 0;
    for(let i = 0; i <= 9; i++){
        if (!numbers.includes(i)) { //.includes : 메서드는 배열이 특정 요소를 포함하고 있는지 판별합니다.
            answer += i;
        }
    }
    return answer;
}

총평 : includes이용하지 않고 하는 방법을 처음에 했지만, 너무 복잡해서 includes 메서드를 쓰는 방식을 체택하였다.
