수박수박수박수박수박수?

길이가 n이고, "수박수박수박수...."와 같은 패턴을 유지하는 문자열을 리턴하는 함수, solution을 완성하세요.
예를들어 n이 4이면 "수박수박"을 리턴하고 3이라면 "수박수"를 리턴하면 됩니다.

n은 길이 10,000이하인 자연수입니다.

입출력 예

n return
3 "수박수"
4 "수박수박" 
function solution(n) {
    var answer = '';
    for(let i = 1; i<=n; i++ ) {
        if(i % 2 == 0) answer += '박'; else answer += '수';
    }
    return answer;
}

총평 : 짝수면 박 홀수면 수를 반환하게했습니다.
