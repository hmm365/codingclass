  • 07. 숫자 메서드 : Number.isNaN : 어떤 값이 NaN인지 판별합니다.
  • 07. 숫자 메서드 : Number.prototype.toFixed : Number 객체를 주어진 digits 만큼의 소수점 이하 자리수를 정확하게 갖는 문자열 표현으로 반환합니다.
