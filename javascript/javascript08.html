  • 03. 데이트 메서드 : Date.parse : 날짜의 문자열 표현을 구문 분석하고 1970년 1월 1일, 00:00:00 UTC 이후의 밀리초 수를 반환합니다.
  • 04. 데이트 메서드 : Date.UTC : 메소드는 생성자와 유사한 매개변수를 허용 Date하지만 UTC로 처리합니다.

  • 08. 데이트 메서드 : Date.prototype.setMonth : 현재 설정된 연도에 따라 지정된 날짜의 월을 설정합니다.
  • 08. 데이트 메서드 : Date.prototype.getDate : 주어진 날짜의 현지 시간 기준 일을 반환합니다.

  • 11. 데이트 메서드 : Date.prototype.getDay : 현지 시간에 따라 지정된 날짜의 요일을 반환합니다. 여기서 0은 일요일을 나타냅니다.

  • 17. 데이트 메서드 : Date.prototype.setSeconds : 현지 시간에 따라 지정된 날짜의 초를 설정합니다.

  • 19. 데이트 메서드 : Date.prototype.setMilliseconds : 현지 시간에 따라 지정된 날짜의 밀리초를 설정합니다.
